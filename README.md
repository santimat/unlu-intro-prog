# Ejercicios de laboratorio practico de Programación en Python

Este repositorio contiene los ejercicios realizados durante el laboratorio práctico de la materia Introducción a la programación de la Licenciatura en Sistemas de Información de la UNLU `(Universidad Nacional de Lujan)`. Cada ejercicio está organizado en su propia carpeta, con el código fuente y cualquier recurso adicional necesario para su ejecución.
